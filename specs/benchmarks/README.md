# NFL Benchmarks

Benchmark requirements and measurement methodology for NFL.

## Benchmark categories

- `throughput.md` — sequential scan and decode bandwidth targets
- `latency.md` — indexed lookup and query latency targets
- `scalability.md` — multi-core concurrency and lock-free scaling
- `vector-search.md` — ANN and similarity search performance targets
- `io-patterns.md` — efficient IO pattern and direct IO validation
