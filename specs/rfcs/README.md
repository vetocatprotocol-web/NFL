# NFL RFCs

Formal design proposals for NFL subsystems.

## Available RFCs

- `0001-format-architecture.md` — immutable file and segment architecture
- `0002-io-and-mmap-model.md` — I/O model, mmap-first semantics, and async/direct IO design
- `0003-vector-tensor-storage.md` — vector, tensor, and quantized storage models
- `0004-indexing-and-search.md` — index and ANN search architecture
- `0005-security-integrity.md` — encryption, checksums, and integrity boundaries
